# Explore React Form

A React practice project exploring different ways to handle forms and manage state with Context API.

## Topics Covered

- Access form data using `onSubmit` and `e.target`
- Use form action, `FormData`, and controlled components
- Controlled vs uncontrolled ways to collect form data
- Custom hook to reduce code duplication
- Product form with table display and error handling
- Family component tree structure
- Context API — basic and advanced usage

## Tech Used

React · Vite · JavaScript · CSS
